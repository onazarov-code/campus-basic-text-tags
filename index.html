Basic HTML Text Tags

Marking up Text in Blocks

When text appears in a block with several lines, it is a paragraph. Paragraphs can contain important words, emphasized phrases, and inline code like <code>.

For short poems or preformatted lines use line breaks (<br>) or a dedicated element — but prefer CSS for layout.

Excerpt from The Raven — Edgar Allan Poe

Once upon a midnight dreary, while I pondered, weak and weary,
Over many a quaint and curious volume of forgotten lore—
While I nodded, nearly napping, suddenly there came a tapping,
“’Tis some visitor,” I muttered, “tapping at my chamber door—”

Marking Up Text to Convey Meaning

Use the correct tag to show meaning. For example, highlight search terms with focus area or show small emphasis with bold and italic.

Keep content accessible and well-structured for screen readers and good HTML practice.
