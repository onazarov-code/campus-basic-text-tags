Common HTML Text Elements

Writing Text in Paragraphs

When text appears in a block with several lines, it’s treated as a paragraph. Paragraphs can hold phrases or words that are emphasized or styled in different ways. They can also include links or other inline elements — but not headings. Paragraphs are considered block-level elements.

Sometimes, you might need to display text that needs a specific layout, like poetry. In that case, you can use line break tags to control where the lines end. These tags should only be used for special formatting, such as poems or addresses. Avoid adding line breaks inside normal paragraphs or forms — use CSS instead for proper styling.

Keep in mind that tags like <strong> and <em> are used to express meaning in text, not just to make words look bold or italicized.

Using Text to Communicate Meaning

Sometimes, only part of a sentence — a few words or letters — needs special meaning. In those cases, it’s best to choose the most descriptive HTML tag for the purpose.

For instance, on a page that displays search results, you may want to highlight words that match the user’s search. If someone searched for “focus area,” that phrase should appear highlighted when shown in the results.

Earlier, we mentioned the <b> and <i> tags, but not <u>. The underline tag is less common because it can make text look like a hyperlink. A better use is marking spelling or typing mistakes.
