Project: Marking Up My Content

Follow these steps to create and mark up your HTML file in VS Code.

Create the file

Open VS Code and create a new file using the keyboard shortcut Control/Command + N.
Save the file as index.html in your project folder.

Make sure you add the basic HTML skeleton (doctype, html, head, body). Use the meta author tag with your name. Author: Orzu Nazarov.

Under your title and other meta tags, link to the stylesheet with styles.css. Example: the <link> element in the head.

Paste your content

Open the text document that contains your content. Select all with Control/Command + A, copy with Control/Command + C, then paste into the <body> of your HTML file.

Now mark up the paragraphs using semantic tags: wrap the title in an <h1>, group related material in <section> elements, and use emphasis or italics where appropriate to show importance or tone.

Remember accessibility: use headings in order and meaningful text in links and images. Accessibility matters.

Final formatting & small markup details

Use inline <code> tags to show filenames or commands in documentation. Use highlight when you need to call attention to specific words.

If your original text includes citations, add them with a cite element or create a reference list in the footer.
